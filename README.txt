Files/folders:
- ms.tex: main .tex file
- ch1.tex, ch2.tex, ch3.tex, ch4.tex: subsidiary .tex files called by ms.tex
- preamble.tex: subsidiary .tex file called by ms.tex
- bibl.bib: bibliographic repository, called by ms.tex
- COPYING.txt: license information.
- Sinander_mechanism_design_notes.pdf: PDF lecture notes compiled on my machine.

To produce PDF lecture notes, compile ms.tex using pdfLaTeX and Biber.
